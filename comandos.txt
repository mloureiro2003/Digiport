git init     Inicia um repositorio Git local
git status   Mostra os arquivos modificados
git add -nome arquivo   Adiciona um arquivo ao staging
git add .   Adiciona todos os arquivos
git commit -m "mensagem"    Cria um commit com uma mensagem 
git log     Mostra o historico de commits
git clone <url>     Clona um repositorio remoto
git push    Envia os commits locais pro Github (ou outro remoto)
git pull    Atualiza o repositoio local com base no remoto
